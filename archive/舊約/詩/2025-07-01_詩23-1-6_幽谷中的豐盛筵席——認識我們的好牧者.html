詩 23:1-6
幽谷中的豐盛筵席
認識我們的好牧者

「耶和華是我的牧者，我必不至缺乏。」

— 詩篇 23:1

會前禱告

慈愛的天父，我們滿心感謝祢。祢是那為羊捨命的好牧者，無論我們身處青草地、可安歇的水邊，或是死蔭的幽谷，祢的杖和竿永遠引導、保護著我們。求祢的聖靈今天向我們的心說話，使我們在這變動不安的世界中，靈魂得著甦醒。幫助我們放下憑靠自己掙扎的重擔，單單仰望祢的供應與同在，深信祢的恩惠與慈愛必一生一世追隨著我們。奉主耶穌基督的聖名祈求，阿們。

第一段深度解析

青草地上的滿足與甦醒
詩 23:1-3a

「耶和華是我的牧者，我必不至缺乏。他使我躺臥在青草地上，領我在可安歇的水邊。他使我的靈魂甦醒。」

關鍵原文 / Keyword
缺乏 (chasar / חָסֵר)
「減少、枯竭、匱乏」。不僅是物質上的缺少，更是生命狀態中的空虛。
甦醒 (shuv / שׁוּב)
「轉回、恢復、帶回」。迷失或疲憊的羊被找回，生命力得到完全更新。
關鍵解析 / Insight

為什麼在物質豐裕的現代，我們卻常感到靈魂的匱乏？如何讓神真正成為我們的「牧者」？

我們的匱乏感往往來自於將受造物當作滿足的源頭。當大衛宣告「耶和華是我的牧者」時，他交出了生命的主權。羊無法自己尋找青草地，唯有徹底信任牧者的帶領，承認自己的有限，我們才能在神裡面找到「不至缺乏」的真正安息。

提摩太·凱勒 (Timothy Keller)

「如果你以為擁有耶穌和擁有其他東西才能使你滿足，那麼你還沒有把耶穌當作真正的牧者；如果耶穌是你所擁有的一切，你就會發現，祂就是你所需要的一切。」

第二段深度解析

死蔭幽谷中的無懼與同在
詩 23:3b-4

「為自己的名引導我走義路。我雖然行過死蔭的幽谷，也不怕遭害，因為你與我同在；你的杖，你的竿，都安慰我。」

關鍵原文 / Keyword
死蔭 (tsalmaveth / צַלְמָוֶת)
「陰影」與「死亡」組成，指極度的黑暗、深沉的危險。
安慰 (nacham / נָחַם)
「深吸一口氣、憐憫」。牧者的杖和竿帶給羊群深刻的安全感。
關鍵解析 / Insight

既然神引導我們走「義路」，為什麼這條路上還會有「死蔭的幽谷」？

幽谷並非神引導失敗，有時正是通往高山豐盛牧場的必經之路。大衛在經文中的代名詞從「祂」轉變成了「祢」。在青草地上，我們談論神（祂）；但在死蔭幽谷中，我們經歷神（祢）。苦難逼使我們與神面對面。

司布真 (Charles Spurgeon)

「我們在幽谷中所學到的，比在山頂上學到的更多；神在黑夜中賜下的詩歌，比在白晝時更加甜美。」

第三段深度解析

敵人面前的豐盛與永恆的歸宿
詩 23:5-6

「在我敵人面前，你為我擺設筵席；你用油膏了我的頭，使我的福杯滿溢。我一生一世必有恩惠慈愛隨著我；我且要住在耶和華的殿中，直到永遠。」

關鍵原文 / Keyword
隨著 (radaph / רָדַף)
「追趕、追擊」。神的恩典是主動、猛烈地「追趕」著我們。
慈愛 (chesed / חֶסֶד)
神堅定不移、立約的愛，不受人類軟弱的影響。
關鍵解析 / Insight

當生活充滿敵對、攻擊與壓力時，神如何為我們「擺設筵席」？

神沒有立刻把敵人消滅。神選擇在敵人的注視下為他擺設豐盛筵席，代表極大的安全感與確據。這告訴我們，內在的平安與喜樂不需要等到風暴平息，在風暴中心，因著神的恩惠追趕，我們已經可以享受滿溢的福杯。

畢德生 (Eugene Peterson)

「神的恩惠與慈愛不是消極地跟隨在我們身後，而是如獵犬般積極地追尋我們，直到把我們追入祂永恆的家中。」

講道應用方向

  • 01

    檢視主權與知足：誠實面對生命中的「缺乏感」，學習將生活主權交還給牧者。

  • 02

    轉換苦難的視角：身處幽谷時，停止追問「為什麼」，轉而緊抓神的「同在」。

  • 03

    在風暴中享受筵席：不需等到問題解決才感恩，刻意數算神那主動「追趕」我們的慈愛。

講章文稿

詩23:1-6: 幽谷中的豐盛筵席——認識我們的好牧者

弟兄姐妹們，平安！我們今天所處的世界充滿了變動與不安，無論是經濟的壓力、健康的危機，還是人際關係的破碎，都時常讓我們感到疲憊與恐懼。然而，在聖經中最為人熟知的詩篇第23篇中，大衛王向我們描繪了一幅充滿安息與盼望的圖畫。今天的講道，我們要一起來認識這位在變動世界中永不改變的牧者。

青草地上的滿足與甦醒

「耶和華是我的牧者，我必不至缺乏。」(詩篇 23:1)

這是整篇詩篇的金句，也是大衛生命的核心宣告。希伯來文中的「缺乏」，指的不僅是物質的沒有，更是靈魂的空洞。為什麼我們常感到匱乏？因為我們常讓金錢、工作或別人的眼光來做我們的牧者。羊群極易受驚、毫無防禦能力，牠們無法自己找到青草地，除非牧人親自帶領，排除一切恐懼與飢餓的干擾，牠們才能「躺臥」。當我們願意放下自己的掙扎，讓神引導我們到可安歇的水邊，祂必使我們的靈魂「甦醒」，也就是將我們迷失的生命重新尋回並修復。

死蔭幽谷中的無懼與同在

「我雖然行過死蔭的幽谷，也不怕遭害，因為你與我同在。」

幽谷不是神引導失敗的結果，它往往是通往更高豐盛之地的必經之路。請注意，在這節經文裡，大衛的稱呼改變了。在青草地上，他說「祂」使我躺臥；但在幽谷中，他說「祢」與我同在。苦難與黑暗逼使我們從客觀地「談論神」，轉變為主觀地「經歷神」。在幽谷中，神不再是遙遠的「祂」，而是緊緊牽著我們手的「祢」。牧羊人的「杖」擊退野獸，「竿」把跌倒的羊勾回來，這帶給羊群極大的安慰。

敵人面前的豐盛與永恆的歸宿

「在我敵人面前，你為我擺設筵席...我一生一世必有恩惠慈愛隨著我。」

當生活充滿敵對、攻擊與壓力時，神沒有立刻把敵人變不見。相反地，祂在敵人的注視下，為你擺設筵席！這是一種極大的得勝與安穩。真正的平安不是沒有風暴，而是在風暴中心可以坐下來享受神滿溢的福杯。更奇妙的是大衛用了一個詞：「隨著」。在原文裡，這是軍隊「追趕、追擊」的意思。神的恩惠與慈愛不是慢吞吞地跟在我們後面，而是像獵犬一樣，主動、猛烈地追趕著我們，直到把我們追進耶和華的殿中，直到永遠。

願我們都能深深經歷「耶和華是我的牧者，我必不至缺乏」的真實，一生一世安居在祂的殿中。
