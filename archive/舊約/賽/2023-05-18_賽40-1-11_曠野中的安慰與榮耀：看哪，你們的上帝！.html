開啟祈禱

「親愛的天父，我們來到祢的面前，承認我們時常行走在生命的曠野中，感到疲乏、軟弱與迷失。主啊，感謝祢賜下安慰的應許，宣告祢的刑罰已經滿了，祢的恩典已經臨到。求祢的聖靈今天開啟我們的心，讓我們在瞬息萬變的世界中，看見唯有祢的話語永遠立定；在我們無助的時候，看見祢不僅是大能的得勝者，更是將我們抱在懷中、溫柔引導的牧者。求祢除去我們心中的高低不平，為祢預備道路，使我們能全心歡呼：『看哪，我們的上帝！』奉主耶穌基督的聖名禱告，阿們。」

賽 40:9-11

深度解析：曠野中的呼聲

「你們的神說：你們要安慰，安慰我的百姓。要對耶路撒冷說安慰的話，又向她宣告說，她爭戰的日子已滿了；她的罪孽原諒了；她為自己的一切罪，從耶和華手中加倍受罰...」

關鍵原文 / Keyword

  • 安慰 (נָחַם / Nacham)

    帶有強烈情感，原意包含「深呼吸」、「嘆息」。上帝迫切命令要將這份解除痛苦的力量帶給百姓。

  • 預備 (פָּנָה / Panah)

    原意為「轉向」、「清理」。在古代，這是君王出巡前先鋒部隊的職責。

關鍵問題與解析 / Insight

當生命陷入荒涼的「曠野」時，我們該如何經歷上帝的安慰？

上帝的安慰建立在「罪得赦免」與「神親自臨在」的基礎上。我們必須「預備道路」——削平驕傲、填補絕望、修正虛謊。當心思意念被修剪，神的榮耀自然顯現。

例證：集中營中的安慰

彭柯麗在拉文斯布呂克集中營的黑暗中，透過聖經經歷了上帝的真實同在。她們在心裡的曠野為神預備了道路，神的榮耀就在跳蚤滿地的牢房中顯現。

「上帝在我們的歡樂中對我們微語，但在我們的痛苦中大聲疾呼。」—— 提摩太·凱勒

深度解析：永不改變的應許

「凡有血氣的盡都如草；他的美容都像野地的花... 草必枯乾，花必凋殘，惟有我們神的話必永遠立定。」

關鍵原文 / Keyword

  • 美容 (חֶסֶד / Chesed)

    此處指人的良善、成就。在上帝的永恆面前，這一切都脆弱如花。

  • 立定 (קוּם / Qum)

    站立、堅立。表示無論環境如何變遷，上帝的話語將穩固不搖動。

關鍵問題與解析 / Insight

在瞬息萬變的世代，什麼是真正的安全感來源？

人常將安全感建立在「美容」上，但這受制於時間。唯有將根基建立在「神的話語」上，才能擁有不受風雨動搖的永恆安全感。

例證：燈塔與別墅

豪華別墅在颶風中瞬間瓦解，但深深扎根磐石的燈塔卻能屹立不搖並指引方向。這就是神的話與世俗成就的區別。

「唯有上帝的話語，是我們在風暴中唯一的錨。」—— 馬丁·路德

深度解析：大能與溫柔的牧者

「主耶和華必像大能者臨到... 他必像牧人牧養自己的羊群，用膀臂聚集羊羔抱在懷中，慢慢引導那乳養小羊的。」

關鍵原文 / Keyword

  • 報好信息 (בָּשַׂר / Basar)

    宣揚勝利，這正是新約「福音」（Gospel）一詞的舊約根源。

  • 牧養 (רָעָה / Ra'ah)

    包含餵養、看顧、保護、引導、作伴等多重溫柔含義。

關鍵問題與解析 / Insight

威嚴大能與細膩溫柔，這兩者如何同時存在？

上帝是大能者，足以擊敗死亡與罪惡；祂更是牧人，願意體恤我們的軟弱，將羊羔抱在懷中配合我們的腳步「慢慢引導」。

例證：君王的背膀

一位威震四方的國王，回到宮中卻願意趴下讓幼女騎在背上。上帝的大能用來保護，溫柔則保留給祂所愛的兒女。

「祂無限的權能與無限的柔和同時存在。」—— 陶恕

Full Sermon 講章全文

弟兄姊妹們，平安。今天我們要進入以賽亞書第四十章。

如果我們的人生是一幅畫，很多時候這幅畫的背景不是鳥語花香的伊甸園，而是乾旱疲乏的曠野。或許你現在正面臨經濟的壓力、家庭的破碎、疾病的折磨，或者靈性深處的枯竭。我們都在呼喊：出路在哪裡？今天，上帝要透過這段經文，向我們每一位在曠野中的人發出呼喚。

一、 曠野中的呼聲：預備主道，迎接安慰

經文一開始，上帝就用最迫切的語氣說：「你們的神說：你們要安慰，安慰我的百姓。」這裡的「安慰」不是敷衍地拍拍肩膀，而是上帝要親自挪去我們的重擔。

但安慰如何臨到？「在曠野預備耶和華的路，在沙漠地修平我們神的道。」上帝的安慰從來不是降臨在驕傲與虛偽的心裡。提摩太·凱勒牧師曾說：「上帝在我們的歡樂中對我們微語，但在我們的痛苦中大聲疾呼。」當我們願意在人生的曠野中放下掌控，「耶和華的榮耀必然顯現」。

二、 永不改變的應許：草必枯乾，主道永存

先知把我們的眼光轉向永恆：「凡有血氣的盡都如草...惟有我們神的話必永遠立定。」

這是一個震撼的對比。我們花了多少精力追求那些「必凋殘的花」？財富、地位、健康，在上帝的氣息面前不堪一擊。馬丁·路德說過：「唯有上帝的話語，是我們在風暴中唯一的錨。」只要你手中還有上帝的應許，你就擁有了重新站立的本錢。因為草必枯乾，花必凋殘，惟有我們神的話必永遠立定！

三、 大能與溫柔的牧者：看哪，你們的神

先知呼籲我們：「揚聲不要懼怕，對猶大的城邑說：看哪，你們的神！」這是舊約裡的「福音」！

祂是怎樣的神？祂是「大能者」，膀臂必為祂掌權；但祂也「必像牧人牧養自己的羊群」。請注意奇妙的對比：同一雙膀臂，在第10節是擊退仇敵的大能，在第11節卻是將羊羔抱在懷中的溫柔。祂體恤你的軟弱，祂會「慢慢引導」你。

今天，讓我們全然降服在祂面前，歡呼說：「看哪！這是我的神！」阿們。
